## FineVQ 单图推理脚本
- 在 `quality_test/FineVQ/scripts/run.py` 中编写了最简单图质量推理逻辑
- 模型：`IntMeGroup/FineVQ_score`（基于 InternVL + SlowFast 架构）
- 核心发现：`infer.sh` 对应的 inference.py 实际上是视频质量评估（LIVE_VQC），模型需要 pixel_values(ViT patches) + pixel_values2(slowfast frames)
- 单图推理的关键：将图片复制 32 帧模拟静态视频输入给 slowfast 模块
- 模型 forward 输出 score1 = mlpscore(last_hidden_states[:,-4,:])，即从 LLM 最后一层倒数第4个 token 的 hidden state 通过 MLP 回归出质量分数
